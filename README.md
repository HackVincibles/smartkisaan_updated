# Smart Kissan - Trust-Enabled Agricultural Marketplace

A unified repository for the Smart Kissan project, integrating a blockchain-based trust engine with a modern full-stack web application.

## Project Structure

- **`/client`**: Vite-based React frontend.
- **`/server`**: Express-based Node.js backend.
- **`/blockchain`**: Smart contracts (Polygon Amoy testnet).

## Getting Started

### 1. Server Setup
```bash
cd server
npm install
# Ensure .env is configured with MONGODB_URI and POLYGON_PRIVATE_KEY
npm run dev
```

### 2. Client Setup
```bash
cd client
npm install
npm run dev
```

## Key Features

- **Blockchain Trust Engine**: Automated escrow and reputation management using smart contracts.
- **AI Quality Grading**: Computer vision for crop grading and shelf-life estimation.
- **Real-time Updates**: Socket.io integration for order tracking and notifications.
- **Secure Payments**: Razorpay integration for seamless transactions.
